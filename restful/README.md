# RESTFUL 

http://www.ruanyifeng.com/blog/2018/10/restful-api-best-practices.html


五种 HTTP 方法，对应 CRUD 操作

    GET：读取（Read）

    POST：新建（Create）

    PUT：更新（Update）

    PATCH：更新（Update），通常是部分更新

    DELETE：删除（Delete）


HTTP 状态码，分成五个类别

    1xx：相关信息

    2xx：操作成功

    3xx：重定向

    4xx：客户端错误

    5xx：服务器错误

200状态码

    GET: 200 OK

    POST: 201 Created

    PUT: 200 OK

    PATCH: 200 OK

    DELETE: 204 No Content
